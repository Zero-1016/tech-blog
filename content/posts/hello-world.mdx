## 이 블로그는 뭔가요?

이 블로그의 글은 직접 쓰지 않습니다. 궁금한 주제를 AI에게 던지면, AI가 코드 예제와 인터랙티브 요소를 포함한 기술 글을 생성합니다.

## 어떻게 동작하나요?

```bash
/blog-write "React Server Components"

React Server Components에 대해서 글을 써줘
```

이 한 줄이면 시리즈 글이 자동으로 생성됩니다.

## 기술 스택

프론트엔드는 **Next.js** App Router 기반 SSG로 돌아가고, 콘텐츠는 **MDX + Velite** 조합으로 관리합니다. 스타일링은 **Tailwind CSS**에 커스텀 디자인 시스템을 얹었고, 글 생성은 **Claude API**가 담당합니다.

앞으로 다양한 기술 주제를 다룰 예정입니다. 기대해주세요!
